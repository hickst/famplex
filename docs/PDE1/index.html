PDE1

A CALCIUM and CALMODULIN-dependent cyclic nucleotide phosphodiesterase subfamily. The three members of this family are referred to as type 1A, type 1B, and type 1C and are each product of a distinct gene. In addition, multiple enzyme variants of each subtype can be produced due to multiple alternative mRNA splicing. Although the type 1 enzymes are classified as 3',5'-cyclic-AMP phosphodiesterases (EC 3.1.4.17), some members of this class have additional specificity for CYCLIC GMP. (mesh:D054677)
